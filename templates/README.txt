# Template Images สำหรับ RedM Fishing Macro
# =============================================
#
# โฟลเดอร์นี้เก็บภาพ template ที่ใช้สำหรับ image detection
#
# วิธีสร้าง template:
# 1. รัน: python fishing_macro.py --capture
#    (หรือดับเบิลคลิก run_macro.bat --capture)
# 2. ทำตามขั้นตอนบนจอ
#
# หรือสร้างเอง:
# 1. เปิดเกม RedM
# 2. Screenshot หน้าจอ (กด Print Screen)
# 3. เปิดใน Paint → Crop เฉพาะส่วนที่ต้องการ
# 4. Save เป็น .png วางในโฟลเดอร์นี้
#
# ไฟล์ที่ต้องมี:
# ─────────────────────────────────────────────
# lure.png          - ไอคอนเหยื่อตกปลาในกระเป๋า
# use_button.png    - ปุ่ม "ใช้" ในแถบ action
# minigame_bar.png  - ข้อความ [ SPACE ] ตอนมินิเกม
# hook_icon.png     - ไอคอนเบ็ดในแถบมินิเกม
# pickup_prompt.png - ข้อความ [ G ] PICKUP
# ─────────────────────────────────────────────
